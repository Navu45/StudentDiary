diary.schedule module¶

diary.schedule.get_schedule_from_mirea()¶

Парсит расписаниие с „https://www.mirea.ru/schedule/“ и преобразует его в json файлы. На выходе получается 3 .xlsx файла и 3 .json файла (нумерация файлов в соответствии с курсом).

diary.schedule.show_schedule_for_day(group, d)¶

Парсит расписаниие json файл и возвращает расписание определенного дня в str.

Параметры:

group (str): студенческая группа d (datetime.datetime): дата

Возвращаемое значение:

schedule (str): расписание на день

Параметры:
